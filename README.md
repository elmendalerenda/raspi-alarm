#Raspi Alarm

An alarm clock system for the raspberry pi
Features:

 - Schedule events from a Google Calendar into cron events
 - Play a song from a playlist when the alarm go off

## Usage

```bash
$> bundle install
```

## Run the tests

For unit tests: 
```bash
$> rake test
```
For all the tests including the integration tests: 
```bash
$> rake all_tests
```
